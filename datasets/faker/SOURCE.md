# Dataset source

Generated locally with Faker on 2026-07-18 using seed `20260718`.

- Generated samples: 1,500 synthetic text records.
- Entities per generated record: person, email, phone, company, address, SSN, credit card, date of birth, and IP address.
- Output: one UTF-8 text document per record plus matching labels in `labels.json`.

The seed makes the dataset reproducible with the installed Faker version. It contains no real personal information.
